#### ~={green}How to use=~ :

Install packages :
```
apt install
```

Downlaod a package without installation :
```
apt install -d
```

Simulate installation :
```
apt install -s
```

Unistall the package :
```
apt remove
```

Remove dependences :
```
apt outoremove
```

Search for package cache :
```
apt-cache search
```

Update distro
```
apt dist-upgrade
```

### ~={red}dpkg=~

Configure the timezone of system :
```
dpkg-reconfigure tzdata
```

See content of package :
```
dpkg --content
```

Status of package :
```
dpkg -s
```

Remove completly a package :
```
dpkg --parge
```

Install package :
```
aptitude install
```
